Navigation

Typing any line of code by itself will add that line of code onto the end of the program. However, typing certain things and hitting enter before entering that line, will result in different things happening.

  • Typing add will result in the next line being run as normal, but ignoring any other shortcuts it might trigger by accident. For example, if you have a variable named insert, and you want to run the line insert += 1, you would first enter add, so that Frost doesn’t think you are trying to insert a line
  • Typing run will start the program. From the console, hit space to return to the coding area. This can be finicky (yay, Scratch!), and you must have the last place you clicked be on the border of the list (or you didn’t click anything after entering run
  • Entering fix n where n is a number. fix n will make the next line you enter replace the nth line
  • To insert the next line at line n instead of at the end of the program, enter insert n before the line
  • To delete line n, enter del n

Expressions

Expressions work the same as they would anywhere else. Currently supported are addition (+), subtraction (-), multiplication (*), division (/), exponentiation (^), and parentheses.

Examples:

  • Addition

print 1 + 2

//prints 3

  • Subtraction

print 4 - 1

//prints 3

Comments

Any line starting with // will be ignored. Example:

var variable = “lol im a variable”

// I do nothing!

Print

The built-in function print adds the input to the console. print can take a string or a number as input.

  • Examples:

print “Banana”

//prints Banana

print 1 + 2

//prints 3

Variables

Variables can store a number or a string, and are accessed using their assigned name. To define a variable, you would use something like this:

var variableName = value

Or, to leave it without a value, you could use this:

var variableName

There are a few ways to reassign a variable:

var x = value

//x is equal to value

x = otherValue

//x is equal to otherValue, and no longer equal to value

variableName += value

//Is equivalent to

variableName = variableName + value

//This also works for subtraction, multiplication, and division

An example might look like this:

var onePlusTwo = 1

onePlusTwo += 2

print onePlusTwo + 3

//prints 6

There are also four built in variables:

  • true returns 1, which is useful for being more explicit in while loops and similar
  • false returns 0, which is helpful for the same reasons as true
  • runTime returns the amount of time since the program started
  • maxLoopTime is the amount of time a loop can run for.

REMINDER: Scratch is not case sensitive! runTime and RuNtImE are treated exactly the same!

Booleans

Booleans are either true or false. If you print one to the console, true will print as 1, and false will print as 0. You can type true or false, or you can use one of the seven comparison operators:

  • == checks if the left and right sides are equal to each other
  • != checks if the left and right sides are not equal to each other
  • > checks if the left side is greater than the right side
  • < checks if the left side is less than than the right side
  • >= checks if the left side is greater than or equal to the right side
  • <= checks if the left side is less than or equal to the right side
  • ! returns true if the right side is false, and false if the right side is true

If Statements

An if statement will check if something is equal to true, and if so, will run the code underneath it. If the boolean is false, it will do nothing, unless you have an else statement underneath it. If you do, then - so long as the provided boolean evaluates to false - it will run the code underneath the else statement. Else statements are completely optional.

  • Examples:

var x = 5

if x >= 4:

        print “If statements are cool, x = “ + x

else:

        print “Math is broken!”

//prints If statements are cool, x = 5

You can also run if statements on one line, provided you only have one line of code for the if statement and one for the else.

var x = 5

if x >= 4: print “If statements are cool, x = “ + x else: print “Math is broken!”

NOTE: Since Scratch makes typing tab characters difficult, it will treat a greater than sign (>) exactly the same at the beginning of a line

While Loops

A while loop will keep running a certain block of code repeatedly, so long as the given condition is true. If the loop is running for longer than three seconds, the program will terminate and return an error. To change this value, change the maxLoopTime variable.

  • Example:

var i = 0

var finished = false

while !finished:

        i += 1

        if i > 4:

                finished = true

        print “i = “ + i

print “Finished! i = “ + i
